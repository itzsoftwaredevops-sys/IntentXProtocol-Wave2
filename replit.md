# IntentX Protocol - Wave2

## Overview
IntentX is an advanced, AI-powered decentralized finance protocol that transforms complex DeFi interactions into simple, natural-language intents. Built on BlockDAG, the platform automates yield strategies, optimizes swaps, reduces gas usage, and delivers intelligent execution with real-time analytics.
