## Open Source Components Used

IntentX builds on these fantastic open source projects:

### Frontend
- **React** - UI library (MIT)
- **TypeScript** - Language (Apache 2.0)
- **Vite** - Build tool (MIT)
- **Shadcn UI** - Component library (MIT)
- **TailwindCSS** - CSS framework (MIT)
- **TanStack Query** - Data fetching (MIT)
- **Lucide Icons** - Icon library (ISC)

### Backend
- **Express** - Web framework (MIT)
- **Zod** - Schema validation (MIT)
- **TypeScript** - Language (Apache 2.0)

### Smart Contracts
- **Solidity** - Smart contract language
- **Hardhat** - Development environment (MIT)
- **OpenZeppelin** - Contract library (MIT)
- **Ethers.js** - Blockchain library (MIT)

## Reporting Issues

Found a bug or have a feature request?

1. Check existing [issues](https://github.com/blockdag/intentx/issues)
2. Create a new issue with:
   - Clear title
   - Description
   - Steps to reproduce (for bugs)
   - Expected behavior
